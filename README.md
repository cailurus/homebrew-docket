# Docket

A lightweight macOS app that groups your dock apps into an iOS-style folder. Less clutter, more focus.

## Install

```bash
brew tap cailurus/docket
brew install --cask docket
```

**Requires macOS 13.0 (Ventura) or later.**

## Update

```bash
brew upgrade --cask docket
```

## Uninstall

```bash
brew uninstall --cask docket
```

## Manual Download

Head to [Releases](https://github.com/cailurus/homebrew-docket/releases) to download the `.dmg` directly.
